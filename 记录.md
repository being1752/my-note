### adb

```shell
adb连接安卓设备
adb connect 192.168.8.117

adb启动shell
adb shell

adb在连接1台设备后，让这台设备安装指定apk
adb install HCDN_HTTPFLV_0.2.20_220_2406171848_xiaodu_release.apk

adb在连接1台设备后，让这台设备启动多子流插件
adb shell am start -n migu.hls.edgelive/com.cmdc.edge.httpflv.TestActivity
adb shell am start -n com.baidu.edge.edgelive/com.cmdc.edge.edgelive.TestActivity

adb卸载指定包
adb uninstall migu.hls.edgelive 
adb uninstall com.baidu.edge.edgelive 

过滤登录信息
logcat | grep requestPluginAuthHandle

多子流日志路径
cd data/user/0/com.baidu.edge.edgelive/cache/

查找版本
pm dump com.baidu.launcher |grep version

查找migu包
pm list package |grep migu
```
